# Changelog

## [0.0.2] - 2026-05-03

- Added configurable pattern array with multiple command fallbacks
- Added path transform options (forwardSlash, fileUrl)
- Added support for UNC paths and file:// URLs
- Dynamic configuration reload on settings change

## [0.0.1] - 2025-01-01

- Initial release
- Regex-based pattern matching for file paths
- DocumentLinkProvider for clickable links in editor
- Windows Explorer and Directory Opus integration
