# Change Log

All notable changes to the "gtfish01_bracket-jumper" extension will be documented in this file.

## [0.2.0] - 2024-01-01

### Changed

1. Enhanced uni-bracket (quotes) detection with context-aware opening/closing heuristics.
2. Added configurable `uniBrackets` setting for characters that serve as both opening and closing brackets.
3. Improved ascend commands to handle uni-brackets with proper pair counting.

## [0.1.0] - 2022-07-03

### Added

1. Initial fork with customized keybinding configurations for bracket navigation.
2. Modified shortcut mappings for Mac and Windows platforms.
