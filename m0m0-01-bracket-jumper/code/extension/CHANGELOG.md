# Change Log

## [1.0.0] - 2026-06-14

### Changed

1. Initial fork with customized keybinding configurations for bracket navigation.
2. Enhanced uni-bracket (quotes) detection with context-aware opening/closing heuristics.
3. Added configurable `uniBrackets` setting for characters that serve as both opening and closing brackets.
4. Improved ascend commands to handle uni-brackets with proper pair counting.
5. Modified shortcut mappings for Mac and Windows platforms.
